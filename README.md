# TMS
> TMS是团队协作web系统(响应式界面设计,移动端适配).

## 具有以下功能:
- 团队协作沟通功能(类似于[slack](https://slack.com/) [bearychat](https://bearychat.com/))
- 国际化翻译管理.
- 博客

### 沟通功能
- 频道(组团沟通)
- 私聊(一对一)
- markdown语法支持(内容排版不再单调)
- @消息 收藏消息 富文本消息目录
- 剪贴板上传图片, 拖拽文件上传(就是这么便捷)
- 邮件通知 桌面通知(不用担心错过什么)
- 热键支持(鼠标点多了也累不是)
- 更多贴心便捷操作等你发现
